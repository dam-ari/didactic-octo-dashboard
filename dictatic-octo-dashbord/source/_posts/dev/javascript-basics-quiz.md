The different data types in JavaScript are:
  - `string`
  - `number` (includes integers, floating-point numbers, `Infinity`, and `NaN`)
  - `boolean`
  - `null`
  - `undefined`
  - `symbol` (unique and immutable identifiers)
  - `bigInt` (for integers of arbitrary length)
